# APNG Generator V122 リリースノート

**リリース日:** 2026年5月30日  
**前バージョン:** V121 → **新バージョン:** V122

---

## 📋 概要

V122では、新規トランジション効果「**まばたきイン**」「**まばたきアウト**」を追加しました。
上下のまぶた（楕円弧の黒い領域）が開閉することで、キャラクター視点での「目を開ける / 閉じる」演出を実現します。
TRPGセッション（特にCoC系シナリオ）での目覚め・気絶・意識回復シーンに最適です。

**主なハイライト:**
- まばたきイン（登場）/ まばたきアウト（退場）の2効果を追加
- 瞬き回数（1回 / 2回 / 3回）で開閉パターンを選択可能
- 閉じ具合に連動するガウシアンブラーで朦朧とした視界を表現
- 人間的な揺らぎ（微動 + イージング）でリアルな瞬き動作を再現

---

## 🆕 新機能

### 1. まばたきイン（登場効果）

> キャラクター視点で「目を開ける」演出。上下のまぶたが開き、画像が現れます。

- まぶたは目の形（アーモンド型の楕円弧）で描画
- 開ききると同時にまぶたが画面外へ退避し、元画像が全面表示される
- 閉じているほど視界がぼやけ、開くにつれてクリアになる

### 2. まばたきアウト（退場効果）

> キャラクター視点で「目を閉じる」演出。上下のまぶたが閉じ、画面が暗転します。

- 気絶、眠り、意識喪失などのシーンに最適
- 最終フレームは完全な暗転（真っ黒）で終了

### 3. 瞬き回数オプション

| 選択肢 | 動作 |
|--------|------|
| 1回 | 微動を伴いながら1回で開く（最もシンプル） |
| 2回 | 薄目 → パチッと閉じ → 開眼（少し不安定な目覚め感） |
| 3回 | パチパチパチ → やっと開く（朦朧・意識混濁状態の表現） |

## 🛠 技術的な変更点

- `calcBlinkOpenRatio()`: 瞬き回数に応じた openRatio タイムライン計算（微動 tremor + easeInOutCubic イージング）
- `drawBlinkFrame()`: 楕円弧によるまぶた描画 + 閉じ具合連動ブラー + 境界の影ぼかし
- プレビュー / サイズ推定 / APNG生成 / CSSプレビューの全レンダリングパスに対応
- まばたき効果選択時のデフォルト再生速度を 0.5x（ゆっくり）に設定
